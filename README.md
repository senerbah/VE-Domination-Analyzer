# VE-Domination Analyzer

Experimental framework for vertex-edge domination analysis.

## Features

- γve computation
- Γve computation
- Well-ve-dominated detection
- Minimal VE-dominating set enumeration
- Cycle pattern analysis

## Verified Results

For cycles:

γve(Cn) = ceil(n/4)

Γve(Cn) = floor(2n/5)

Well-ve-dominated cycles:

C3, C4, C5, C6, C7, C9
